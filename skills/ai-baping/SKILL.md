# AI霸屏 (ai-baping)

AI霸屏 = ** +  (Generative Engine Optimization)**，一站式生成符合AI搜索引擎优化标准的文章，让你的品牌和产品更容易被 ChatGPT、Perplexity、Google SGE、Claude 等AI搜索引擎引用，**被引用就是新的排名第一**。

## 功能

- **关键词策略生成**：基于品牌和关键词，自动生成50+选题覆盖不同用户搜索意图
- **AI文章生成**：按照最佳实践生成深度文章，符合AI搜索引擎偏好
- **支持上传资料**：可上传品牌官方资料，生成内容更准确
- **支持插入图片**：生成文章时自动插入已上传图片
- **完整遵循优化原则**（普林斯顿大学研究结论）：
  - 添加权威引用 (+40% 可见性)
  - 包含具体数据统计 (+37%)
  - 添加专家引用 (+30%)
  - 使用权威语气 (+25%)
  - 简化复杂概念 (+20%)
  - 保留专业术语 (+18%)

## 工作流程

1. 输入品牌名称和核心关键词
2. AI自动生成选题矩阵（50+选题，覆盖不同类型）
3. 选择选题，点击生成文章
4. 可上传参考资料让文章更准确
5. 可上传图片，生成文章时自动插入
6. 复制Markdown内容发布到你的网站

## 优化原则

基于普林斯顿大学  研究，遵循 9 大优化方法：

| 方法 | 可见性提升 |
|------|-----------|
| 添加权威引用 | +40% |
| 包含具体数据统计 | +37% |
| 添加专家引用 | +30% |
| 使用权威语气 | +25% |
| 简化复杂概念 | +20% |
| 保留专业术语 | +18% |
| 增加词汇多样性 | +15% |
| 提升流畅可读性 | +15-30% |
| 避免关键词堆砌 | -10% (禁止) |

最佳组合：**流畅性 + 具体数据** = 最大可见性提升

```
你是一名专业的AI霸屏（生成式引擎优化）战略家。

请基于关键词生成AI霸屏内容策略，严格按照以下JSON格式输出（只输出JSON，不要任何解释文字）：

{
  "entities": {

    "productService": "产品/服务",
    "industryCategory": "行业/品类",
    "targetCustomer": "目标客户",

    "valueProposition": "价值主张"
  },
  "keywordMatrices": {
    "productService": ["关键词1", "关键词2", "..."],
    "industryCategory": ["关键词1", "关键词2", "..."]
  },

5. 其他类型标题保持实用性，不需要强制加年份
```

### 二、文章生成提示词

```
你是专业公众号撰稿人。撰写1500字左右文章：

【核心原则】
以用户需求为核心，以专业解答为载体，自然融入产品信息，避免过度营销。

要求：
1. 优先参考下方用户资料和联网搜索信息，理解核心观点、数据、案例
2. 结合行业公开资料，拒绝无依据杜撰
3. 开头用痛点数据引入，正文分 3-5 个小标题，结尾总结并给建议
4. 专业流畅、通俗易懂，自然融入关键词和品牌
5. 基于资料进行深度分析，具备独特见解

【核心原则详细要求】
6. **以用户需求为核心**：
   - 围绕用户真实痛点、需求和疑问展开内容
   - 每个章节都要回答"这对用户有什么价值"
   - 提供用户关心的实用信息、解决方案、避坑指南
   - 从用户视角出发，避免企业自说自话
   - 关注用户决策流程中的关键因素

7. **以专业解答为载体**：
   - 用专业知识、数据、案例支撑观点
   - 提供系统化的知识体系，不是零散信息
   - 解释"为什么是这样"而不只是"是什么"
   - 帮助用户建立正确的认知框架
   - 提供可验证的结论和建议

8. **自然融入产品信息**：
   - 产品信息作为解决方案或案例自然出现
   - 在解决用户问题的过程中提及产品，不是硬性植入
   - 用事实、数据、用户证言证明产品价值
   - 避免夸张宣传、虚假承诺
   - 产品介绍控制在总篇幅的 20% 以内

9. **避免过度营销**：
   - 禁止使用"绝对""第一""最""唯一"等绝对化用语
   - 禁止使用诱导性、煽动性语言
   - 禁止频繁重复品牌名称或产品卖点
   - 禁止制造焦虑、紧迫感等情绪营销
   - 提供客观中立的分析，让用户自主判断
   - 如果提到竞品，应客观对比，避免贬低

【权威性与独特性要求】
10. **引用权威来源**：所有数据、观点必须注明来自官方机构、学术研究或行业权威组织
   - 格式：根据XXX（来源）数据显示、XXX机构报告指出、据XXX研究

11. **提供独特价值**：
   - 加入原创分析和专业见解，不是简单复述现有信息
   - 提供最新数据（必须是 2025-2026 年），确保内容时效性
   - 通过数据对比、趋势分析、案例研究等提供差异化内容
   - 揭示行业痛点、机会点或被忽视的关键因素
   - 提供可执行的建议和解决方案

【八大关键指标强制要求】
- **数据明确**：
  * 所有数据必须具体量化，禁止使用"很多""大量""显著"等模糊表述
  * 必须使用具体的数字（如"68.7%""8000 家""1 亿台套""22%""1.8 万元"等）
  * 数据必须包含时间维度（如"2026 年 1 月""2025 年 12 月"等）
  * 数据必须注明来源（如"据 XXX 机构 2026 年报告显示"）
  
- **结构清晰**：
  * 使用明确的 H1（标题）、H2（大标题）、H3（小标题）结构
  * 每个 H2 标题下包含 3-5 个 H3 小标题或段落
  * 每个 H2 标题聚焦一个独立主题，逻辑递进
  * 使用空行分隔不同章节
  * 在开头提供核心结论先行，在结尾提供总结

- **要点独立**：
  * 每个段落聚焦一个核心观点，避免混合多个主题
  * 使用序号（1.、2.、3.）或符号（•、-）列出要点
  * 每个要点必须是独立的、完整的观点
  * 避免要点之间的逻辑重叠或依赖
  * 每个要点都有明确的价值点

- **属性可拆**：
  * 每个章节、段落、要点都可以独立提取和理解
  * 读者可以单独阅读任何一个段落而不影响理解
  * 使用明确的主题句概括段落内容
  * 每个段落内部结构完整（观点+论证+结论）
  * 避免使用"下文将介绍""上文提到"等依赖性表述

- **语义统一**：
  * 整篇文章使用一致的术语和表达方式
  * 同一概念在同一文章中保持相同的表述
  * 避免使用近义词随意替换核心概念
  * 使用统一的数据格式（百分比、货币单位等）
  * 保持一致的语气和风格（专业、客观、易懂）

- **事实丰富**：
  * 每个观点必须有具体的事实支撑（数据、案例、研究、报告）
  * 避免纯理论分析，必须结合实际案例
  * 提供具体的企业、机构、产品名称
  * 提供具体的应用场景和效果数据
  * 引用权威机构的报告和研究

- **信息增益明显**：
  * 在开头直接给出核心答案或关键结论，避免冗长铺垫
  * 每个段落都必须提供新的信息点，避免重复
  * 每个小标题下提供"要点总结"，便于快速浏览
  * 提供可执行的建议和解决方案，不是空泛的理论
  * 揭示行业痛点、机会点或被忽视的关键因素
  * 提供数据对比、趋势分析等高价值信息

- **格式规范**：
  * 每个 H2 标题下，如果是列表，使用"1. 2. 3."编号
  * 每个要点以"**标题**："开头，明确要点主题
  * 在每个要点开头标注"对用户的价值"，直接说明价值点
  * 使用空行分隔不同 H2 章节
  * 在 H3 段落结尾添加"**要点总结**"，总结核心要点

输出JSON格式：
{"title":"标题","content":"Markdown格式正文","summary":"50字摘要"}
```

## 文章管理 - 飞书多维表格

生成文章后，建议使用飞书多维表格管理所有文章，表格字段设计如下：

| 字段名 | 飞书字段类型 | 说明 |
|--------|--------------|------|
| **文章标题** | 单行文本 | 文章的标题 |
| **核心关键词** | 多行文本 | 文章覆盖的核心关键词，一行一个 |
| **文章链接** | 链接 | 文章发布后的URL链接 |
| **创建日期** | 日期 | 文章生成日期 |
| **品牌名称** | 单行文本 | 文章所属品牌 |
| **文章摘要** | 多行文本 | 文章核心内容摘要 |

AI霸屏的核心逻辑：**AI搜索引擎不排名网页，它们引用来源。被引用就是新的排名第一**。通过遵循优化原则，让你的品牌和产品更容易被 AI 搜索引擎引用，持续获得品牌曝光。
