# Executive Reminder Patterns

Use these patterns when the user wants a sharper, more polished boss-facing agenda.

## Morning Brief Template

```md
Main focus:
<One sentence on the make-or-break issue today>

Priority order:
1. <Item that requires the boss personally>
2. <High-leverage progress item>
3. <Important item whose time can still be compressed>

Key time points:
- <time> <event>; goal: <outcome>
- <time> <event>; pre-check: <materials / numbers / people>

Risk note:
- <risk>; trigger: <what causes the problem>; recommended move: <immediate action>
```

## Meeting Prep Reminder

Use before board meetings, investor calls, client negotiations, hiring interviews, or executive 1:1s.

```md
Meeting objective:
<What result must come out of this meeting>

Confirm before the meeting:
- <number / file / message>
- <possible disagreement>
- <who needs to be aligned first>

Watch during the meeting:
- <key issue 1>
- <key issue 2>

Immediate follow-up:
- <who does what next>
```

## Travel Day Brief

Use when the boss has cross-city trips, airport transfers, roadshows, or dense external meetings.

Focus on:

- Transit buffers and hard departure times
- Which approvals can be remote
- Which calls can move to the car or airport
- Which materials must be downloadable offline
- The one meeting that cannot be compromised even if the day slips

## Evening Rollover

Use when the user wants a close-of-day summary and tomorrow setup.

```md
Close of day:
- Done: <key items truly completed today>
- Not done: <items that must roll to tomorrow>

Tomorrow's first priority:
- <the first thing to protect tomorrow morning>

Delegate tonight:
- <who should take what, with what result, by when>
```

## Triage Rules for Overloaded Days

When the calendar is too full, recommend action instead of passively listing all items.

Use this order:

1. Keep external high-stakes commitments
2. Keep executive-only approvals and decisions
3. Keep revenue, cash, compliance, and key people items
4. Compress internal status meetings
5. Delegate low-leverage internal follow-ups
6. Move anything that is informational but not decision-relevant

## Better Boss-Facing Phrasing

Prefer:

- "Must be decided today"
- "If delayed, the impact will show up in"
- "Recommend handling personally"
- "Recommend delegating to"
- "Confirm before the meeting"
- "This should not be missed because"

Avoid:

- "Feels important"
- "Take a look if you have time"
- "Probably fine"
- "Can sit for now"
