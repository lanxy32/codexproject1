# Course Image Framework Library

This file summarizes the AI course images already provided earlier and turns them into reusable built-in framework samples.

Use this as the default structural reference library before asking the user to resend any old course images.

Only stop relying on this file when the user explicitly says the course-reference images are updated or should be replaced.

## How to choose between the three

Use `专题课包菜单框架` when:

- the boss wants several topic options
- the proposal is still at the productized offering stage

Use `培训课程清单大纲框架` when:

- the proposal needs a more formal curriculum matrix
- the boss wants to show level, module, and detailed contents

Use `7天AI实训排期框架` when:

- the proposal needs a practical schedule
- the output should feel like a real training execution plan

## Reuse rules

- Borrow the frame, not the wording.
- Adapt the theme names to the customer's industry and training goals.
- If the user requests a one-day, two-day, monthly, or quarterly plan, compress or expand these frames without losing their logic.
- Keep the style table-first and business-like.
